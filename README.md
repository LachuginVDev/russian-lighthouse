# Russian Lighthouse

Сайт музыкальной группы: блог, благотворительные сборы, медиа, контакты.

**Backend:** Laravel 12, PHP 8.3, PostgreSQL 16, Redis  
**Frontend:** Vue 3, TypeScript, Vite, Tailwind CSS, Inertia.js  
**Auth:** Laravel Breeze  
**Admin:** Filament  
**Infrastructure:** Docker (nginx, php-fpm, postgres, redis)
