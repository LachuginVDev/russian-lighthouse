# План структуры сайта

## Маршруты и страницы

| URL | Страница | Описание |
|-----|----------|----------|
| `/` | Welcome | Главная: hero, новости, сборы, медиа, миссия |
| `/about` | About | О группе: описание, фото, видео, история |
| `/news` | News/Index | Список новостей/блога, карточки постов |
| `/news/{id}` | News/Show | Страница одного поста |
| `/fundraisers` | Fundraisers/Index | Список активных сборов |
| `/fundraisers/{id}` | Fundraisers/Show | Сбор: описание, прогресс-бар, форма пожертвования |
| `/media` | Media/Index | Медиа: видео, аудио-плеер, галерея |
| `/contacts` | Contacts | Контакты: информация, соцсети, форма обратной связи |

## Переиспользуемые компоненты

- **AppNav** — главное меню (Главная, О группе, Новости, Сборы, Медиа, Контакты + Вход/Регистрация или Панель).
- **PublicLayout** — лейаут публичных страниц: AppNav + контент (slot).
- **Card** — карточка (заголовок, описание, ссылка); для новостей и сборов в списках.
- **ProgressBar** — прогресс сбора (процент, подпись); на главной и на странице сбора.
- **ContactForm** — форма обратной связи (имя, email, сообщение); страница Контакты.
- **Home/** — блоки главной (HeroBlock, NewsSlider, ActiveFundraisers, MediaBlock, MissionBlock) остаются; при необходимости частично переиспользуются на других страницах (например MediaBlock → медиа-секция на О группе).

## Соответствие ТЗ

- **3.1 Главная** — Welcome + блоки (hero, слайдер новостей, сборы, медиа, миссия).
- **3.2 О группе** — About: описание, фото, видео, история (секции на одной странице).
- **3.3 Новости** — News/Index (карточки), News/Show (один пост).
- **3.4 Сборы** — Fundraisers/Index, Fundraisers/Show (прогресс, форма пожертвования).
- **3.5 Медиа** — Media/Index: видео, аудио, галерея.
- **3.6 Контакты** — Contacts: контакты, соцсети, форма обратной связи.

Данные пока заглушки/пропсы; бэкенд (модели, API) — отдельный этап.
